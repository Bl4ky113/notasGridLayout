Que es Grid

Grid Layout o solamente Grid, es un sistema de diseño de páginas web orientado a diseño en 2 dimensiones, es decir hacía arriba y abajo, X y Y, entre otros. Esta forma de diseñar esta en un Grid o rejilla, donde se tiene diferentes celdas, columnas y filas. Siendo parecidas a una tabla, pero muchisimo mejor. De hecho Grid viene a ser un excelente o la mejor forma de diseñar sitios web. Pasando de las tablas, los Floats, posicionar diferentes inline-blocks y Flexbox, siendo este orientado mucho a diseños horizontales o de sola una dimensión. Pero esto no significa que Flexbox sea peor que Grid, de hecho se puede hacer grandiosos trabajos con Grid y Flexbox al mismo tiempo, siendo muy utiles y versatiles.

Soporte Web de Grid

Como Grid es una gran implementación en el desarrollo y diseño web, es muy importante que funcione en todos los dispositivos por igual. En general Grid puede funcionar de forma nativa en todos los dispositivos, computadores y celulares, y en la mayoria de navegadores, obviamente excluyendo a IE. Pero si llega a haber un atributo, o valor no soportado sera escrito acá.

Beneficios de Grid

Grid al nos ahorra el trabajo que eran las tablas del pasado en los 2000's, el tener que hacer mil y un conectores para organizar diferentes Widgets y el usar floats o usar Flexbox de una forma no apropiada. Esto nos volvera nuestra vida mucho más sencillo al momentos de hacer un sitio web con una excelente Semantíca (header, main, aside, nav, footer, article, section). Al poder cumplir esto, nos puede dar también una gran ayuda a el momento de hacer un sitio web Responsive a el dispositivo.

Dejando los benefios en esta lista.

  • Metodos muy sencillos
  • Semantica Web Excelente y Organizada
  • Ayuda y Asistencia a el Diseño Responsive

Terminología de Grid Layout

Es muy importante entender, saber y poder usar los terminos sencillos que se usan en Grid Layout, esto para que nosotros podamos entender y saber en donde y cómo estamos haciendo el diseño.

La Teminología de Grid Layout es:

  • Grid Container
  • Grid Item
  • Grid Line
  • Grid Cell
  • Grid Track
  • Grid Area

Grid Container

Es el elemento padre directo de todo del diseño de Grid Layout y sus elementos, este es puesto por el elemento que tenga aplicado como style:

display: grid;

Grid Item

Son los hijos directos del Grid Container, donde los contenidos de estos no son Grid Items. Lo unico necesario es ser hijo directo del Grid Container.

Grid Line

Es la Linea que divide y forma la estructura del Grid Layout, siendo verticales y horizontales (Columns, Rows).

Grid Cell

Es el espacio entre 2 lineas verticales (Columns) y 2 lineas horizontales (Rows).

Grid Track

Es todo el espacio que puede haber entre 2 lineas, verticales o horizontales.

Grid Area

Es el espacio que hay entre varios Grid Cells

Cloumnas y Filas Fracionarias

Para ocupar todo el Grid Container, podemos hacerlo de una forma medio mala, que es usando los %. O podemos usar una nueva medida que introduce Grid Layout, fr o francción. Divide el Grid Container en partes, se divide cuantas veces usemos la unidad, es decir, si la usamos solo una vez, se divide en 1, si la usamos 2 veces, se divide en 2, y así.

También cuentan las fraciones si se usan por ejemplo: 1fr 2fr 1fr. En este caso se dividiria en 5 partes y se da 2 partes a la columna o fila del centro. 

               #fr-test {
                  display: grid;
                 
                  grid-template-columns: 1fr 2fr 1fr;
                  grid-template-rows: 2fr 1fr 2fr;
               }

Segunda Clase

Puede que al momento de querer que un diseño en Grid Layout sea Responsive, necesitemos que un elemento se adapte a el dispositivo, para hacer que las columnas y filas se adapten, podemos usar esta funcion. 

               minmax(<"Valor Minimo">, <"Valor Máximo">)

MinMax()

Ejemplo de esto: 

               #minmax-test {
                  display: grid;
                 
                  grid-template-columns: 1fr minmax(1em, 2fr) 1fr;
                  grid-template-rows: minmax(1em, 2fr) 1fr minmax(1em, 2fr);

Tercera Clase

Algunas veces al diseñar nuestro Grid Layout, necesitamos repetir algunos valores de varias filas o columnas, para ahorrar este trabajo podemos usar esta funcion:

grid-template-columns: repeat(<"Número de repetir">, <"Medida a Repetir">);

Esta función se puede convinar con otras funciones como minmax(), y ademas se puede aplicar otras columnas o filas ademas de las repetidas.

grid-template-columns: 12em repeat(3, minmax(auto, 12em)) minmax(1fr, auto);

Repeat()

               #repeat-test {
                  display: grid;
                 
                  grid-template-columns: auto repeat(3, 1fr) auto;
               }

Posicionamiento En Grid Layout

En Grid Layout, no podemos posicionar o mover elementos sin un espacio, ya que esto puede hacer que se vea mal el Grid Container y se salgan de lugar los Grid Items. Por Ejemplo:

Bien Hecho

En este ejemplo tenemos una fila más para poder modificar y mover los elementos, aún que se podría ampliar las columnas, es suficiente para manipular los elementos de una manera comoda. 

               #bienPosicionamiento-test {
                  display: grid;
                 
                  grid-template-columns: repeat(3, auto);
                  grid-template-rows: repeat(5, 5em);
                  grid-gap: 1em;
               }

Posicionamiento Con Columnas En Grid Layout

Para Posicionar los elementos en columnas, debemos tener un espacio libre en columnas o filas. Ejemplo: 

               #posicionamientoColumnas_1-test {
                  display: grid;
                 
                  grid-template-columns: repeat(3, 1fr);
                  grid-template-rows: repeat(5, 5em);
                  grid-gap: 1em;
                 
                  #item1 {
                   background-color: darkgreen;
                   grid-column-start: 2;
                   grid-column-end: 3;
                  }
               }

Como podemos ver en el ejemplo, usamos a grid-column-start: y grid-column-end: para mover el Grid Item. Movemos a el Grid Item dependiendo de las Grid Lines, quienes forman los Grid Cells y los Grid Tracks. Estas Lineas se pueden usar tanto valores positivos, contando de izquierda a derecha, ejemplo start: 1, end: 2. O se puede poner valores negativos, que cuenten de izquierda a derecha, ejemplo start: -1, end: -2. Se puede combinar ambos o usar un valor menor en end que en start, ejemplo start: 2, end: 1. Se puede crear nuevas columnas al intentar usar un número mayor a el de Grid Lines. Pero esto puede generar algunos errores en otros Grid Items.

Tabién se puede usar Grid Layout no solo para el posicionamiento, sino para modificar el BoxModel del elemento, Esto lo logramos usando más de dos Grid Cells al determinar cuantas Grid Lines tiene que usar el Grid Item. Ejemplo, para usar dos o más deberiamos tener 3 o más Grid Lines. 

               #posicionamientoColumnas_2-test {
                  display: grid;
                 
                  grid-template-columns: repeat(3, 1fr);
                  grid-template-rows: repeat(5, 5em);
                  grid-gap: 1em;
                  
                  #item1 {
                   background-color: darkgreen;
                   grid-column-start: 1;
                   grid-column-end: 4;
                  }
               }

Posicionamiento Con Filas en Grid Layout

El posicionamiento con Filas en Grid, es lo mismo que en las Columnas, se usan las Grid Lines, se pueden usar valores positivos, negativos y pudiendose combinar entre ambos. Se puede modificar el BoxModel del elemento y su posicionamiento, entre otros. 

               #posicionamientoFilas_1-test {
                  display: grid;
                  
                  grid-template-columns: repeat(4, 1fr);
                  grid-template-rows: repeat(3, 5em);
                  grid-gap: 1em;
                  
                  #item1 {
                   background-color: darkgreen;
                   grid-row-start: 2;
                   grid-row-end: 3;
                  }

Atributo Span

Se puede usar este atributo para usar en las columnas o en las filas, para definir cuantas Grid cells ocupe. 

               #span-test {
                  display: grid;
                 
                  grid-template-columns: repeat(8, 1fr);
                  grid-template-rows: repeat(2, 5em);
                  grid-gap: 1em;
                 
                  #item1 {
                   background-color: darkgreen;
                   grid-column-start: 1;
                   grid-column-end: span 6;
                   
                   grid-row-start: 2;
                   grid-row-end: span 2;
                  }
               }
