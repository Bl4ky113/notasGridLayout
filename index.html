Que es Grid

Grid Layout o solamente Grid, es un sistema de diseño de páginas web orientado a diseño en 2 dimensiones, es decir hacía arriba y abajo, X y Y, entre otros. Esta forma de diseñar esta en un Grid o rejilla, donde se tiene diferentes celdas, columnas y filas. Siendo parecidas a una tabla, pero muchisimo mejor. De hecho Grid viene a ser un excelente o la mejor forma de diseñar sitios web. Pasando de las tablas, los Floats, posicionar diferentes inline-blocks y Flexbox, siendo este orientado mucho a diseños horizontales o de sola una dimensión. Pero esto no significa que Flexbox sea peor que Grid, de hecho se puede hacer grandiosos trabajos con Grid y Flexbox al mismo tiempo, siendo muy utiles y versatiles.

Soporte Web de Grid

Como Grid es una gran implementación en el desarrollo y diseño web, es muy importante que funcione en todos los dispositivos por igual. En general Grid puede funcionar de forma nativa en todos los dispositivos, computadores y celulares, y en la mayoria de navegadores, obviamente excluyendo a IE. Pero si llega a haber un atributo, o valor no soportado sera escrito acá.

Beneficios de Grid

Grid al nos ahorra el trabajo que eran las tablas del pasado en los 2000's, el tener que hacer mil y un conectores para organizar diferentes Widgets y el usar floats o usar Flexbox de una forma no apropiada. Esto nos volvera nuestra vida mucho más sencillo al momentos de hacer un sitio web con una excelente Semantíca (header, main, aside, nav, footer, article, section). Al poder cumplir esto, nos puede dar también una gran ayuda a el momento de hacer un sitio web Responsive a el dispositivo.

Dejando los benefios en esta lista.

  • Metodos muy sencillos
  • Semantica Web Excelente y Organizada
  • Ayuda y Asistencia a el Diseño Responsive

Terminología de Grid Layout

Es muy importante entender, saber y poder usar los terminos sencillos que se usan en Grid Layout, esto para que nosotros podamos entender y saber en donde y cómo estamos haciendo el diseño.

La Teminología de Grid Layout es:

  • Grid Container
  • Grid Item
  • Grid Line
  • Grid Cell
  • Grid Track
  • Grid Area

Grid Container

Es el elemento padre directo de todo del diseño de Grid Layout y sus elementos, este es puesto por el elemento que tenga aplicado como style:

display: grid;

Grid Item

Son los hijos directos del Grid Container, donde los contenidos de estos no son Grid Items. Lo unico necesario es ser hijo directo del Grid Container.

Grid Line

Es la Linea que divide y forma la estructura del Grid Layout, siendo verticales y horizontales (Columns, Rows).

Grid Cell

Es el espacio entre 2 lineas verticales (Columns) y 2 lineas horizontales (Rows).

Grid Track

Es todo el espacio que puede haber entre 2 lineas, verticales o horizontales.

Grid Area

Es el espacio que hay entre varios Grid Cells
